## Firebase 고도화 작업

### Function

- 구글 로그인 환경을 Firebase를 통해 구현
- 로컬 스토리지에 저장한 데이터를 Firebase Storage에 저장하도록 변경
- Firebase Storage에 저장된 데이터를 로드

### 진행상황

- 구글 로그인 버튼 클릭시 팝업창을 통해 구글로그인 할 수 있도록 함.
- 새로고침시 현재 로그인 정보 사라지도록 설정
